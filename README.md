# Browser-Drum-Set

### Description:
Inspired by Wes Bos "Java Script Drum Kit" simple app wrote in pure "Vanilla JS". 
You can play it by mouse or keyboard, also tapping screen of your mobile device. 

### Technology stack:
- HTML5 (ex. flex)
- Sass (ex. nesting, mixins),
- Java Script (core feature),
- ES6 standard (ex. interpolations)
